WhoDo! — Análise de Dúvidas e Perguntas dos Usuários
Após explorar a plataforma WhoDo! (um marketplace de serviços que conecta clientes a profissionais), identifiquei diversas dúvidas que usuários podem ter. Organizei por categorias:

📋 CADASTRO E LOGIN{
    {Qual a diferença entre cadastrar como cliente ou profissional?
       {O cadastro é o mesmo, mas a partir do momento que fizer o cadastro de um serviço sera visto também como Profissional. Isso porque todo profissional também é um cliente.}
    }  
    {Posso ser cliente e profissional ao mesmo tempo?
        {Sim, na verdade todo profissional é cliente de alguém.}
    }
    {O cadastro é realmente gratuito?
        {Sim, naso há custo nenhum para se cadastrar e pesqueisar na plataforma, só havera um custo caso feche um contrato dentro da plataforma, aonde tera a plataforma como intermediadora, garantindo que ambas as partes tenham suporte para efetuar a transação, sendo financeira, contratual e registro de negociação.}
    }
    {Por que preciso informar CPF/CNPJ?
        {Em vários casos sao esse documentos que constam no contrado de prestação de serciços, ou de um MEI, Empresa prestadora de serviços, ou quem contrata precis para registro.}
    }
    {O login com Google e Facebook ainda não funciona?
        {Sim esta funciomando, mas o cadastro completo precisa ir no perfil informar os dados soliciados}
    }
    {Não recebi o e-mail de confirmação, o que faço?
        {Caso entre com um login de conta social, não sera necessário um emai de confirmação, pois a plataforma entenque que seu email ja foi validado pela rede social. Mas caso faça um cadstro utilizando um email válido, e ainda não recebeu seu email, o que vai ocorrer que no seu perfil estara uma mensagem de em Avaliação, até que receba o email e confirme. em caso de não recer o email pode emviar um email para o suporte <email do suprote>solicitando. Mas se realizar um transação dentro da plataforma, e receber uma avaliação com 3 ou mais estrela, sua conta será validade e verificada.}
    }
    {Como altero minha senha?
        {Dentro de Dashboard/configurações tem a opção de alterar senha}
    }
    {Posso mudar meu e-mail cadastrado depois?
        {Seu cadstro é vinculado a seu email principalmente, mas em Dashboard/configurações tem a opção de migrar para nova conta levando todo seu histórico junto, só itens de dados pessoais podem ser alterado na migração, avaliações e historico de atividades na plataforma serão mantidos.}
    }
}

🔍 BUSCA E CONTRATAÇÃO{
    {Como funciona o sistema de busca por proximidade?
        {Ao filtrar o mapa para que os resultados aparecam próximo a localização que vc definiu como seu local, serão listados os profissionais do mais próximos para o mais distante, assim vão aparecer no resultado aqueles que geográficamente esta mais pero de vc.}
    }
    {O que significa "Profissional Verificado"?
        {Quando um profissional ou até mesmo um cliente efetuam um contrato dentro da plataforma isso prova que são "reais" e nao perfil só para enxer o cadastro ou só curioso, isso ajuda a quem vai contratar ou ser contratado a tomar decisão com quem vai negocial.}
    }
    {Como filtro profissionais por preço?
        {Na página de pesquisa, há um filtro,do lado esquerdo, só definir um valor máximo. Se for na versão  mobile o botão estará acima da lista dos profissionais, ao selecionar ele vai se exapnadir monstrando as opções de filtro.}    
    }
    {Posso ver o portfólio do profissional antes de contratar?
        {sim, ao entrar no perfil do profissional, terá todo seu portifólio disponivel para consulta.}
    }
    {Qual a diferença entre "Preço Fixo" e "Sob Orçamento"?
        {Alguns serviço, com por exemplo manicure,ja dá para definir um valor pra contratação, já serviços que dependem de uma visita no local para entender oque será feito, o preço precisa ser conbinado entre as partes.}
    }    
    {Como solicito um orçamento?
        {Diretamente no perfil do profissional, no botão contratar.}
    }
    {Posso negociar o valor do serviço?
        {Sim, você pode utilizar o sistema de mensagens para entrar em contato com o profissional e conversar sobre os valores antes do fechamento do acordo.}
    }
    {Como sei se o profissional está disponível na data que preciso?
        {Você pode enviar uma mensagem diretamente ao profissional para confirmar a disponibilidade da agenda dele, ou combinar isso durante as etapas da solicitação de serviço na tela do pedido.}
    }
}

💼 PARA PROFISSIONAIS (PRESTADORES){
    {Como crio um serviço na plataforma?
        {Acesse o seu Dashboard e vá na aba "Serviços". Lá você poderá cadastrar rapidamente os serviços que oferece, estipular valores e definir a descrição.}
    }
    {Quantos serviços posso cadastrar?
        {Atualmente a plataforma não possui um limite inflexível. Você pode adicionar as especialidades ou variações de serviços de acordo com sua profissão.}
    }
    {Como defino o preço do meu serviço?
        {Ao criar ou editar um serviço, você escolhe se o modal será 'Fixo' (com o preço já estipulado) ou 'Sob Orçamento' (quando requer negociação caso a caso com o cliente).}
    }
    {O que colocar na descrição do serviço?
        {Seja claro sobre o que exatamente você fará, em quanto tempo costuma concluir, se o material está incluso e seu diferencial de mercado. Isso ajuda a convencer o cliente a te escolher!}
    }
    {Como funciona o sistema de seguidores?
        {Clientes que gostarem do seu trabalho podem "Seguir" o seu perfil. Você será salvo na lista de profissionais favoritos deles, facilitando contatos futuros.}
    }
    {Como adiciono fotos ao meu portfólio?
        {Através do seu Dashboard, vá na seção de "Portfólio" e faça o upload das imagens dos trabalhos que deseja exibir na vitrine do seu perfil público.}
    }
    {Posso criar álbuns diferentes no portfólio?
        {A princípio, as fotos do seu portfólio formam uma grande vitrine unificada na sua página de perfil para que todos vejam logo de cara os seus exemplos de trabalho.}
    }
    {Como sou notificado de novos agendamentos?
        {Você receberá um alerta visual dentro da área de notificações do próprio site WhoDo e também ser notificado por e-mail após a solicitação do serviço.}
    }
    {Posso recusar um agendamento?
        {Sim, os pedidos recebidos entram como Solicitações. Você os avalia e decide se deseja Aceitar ou Recusar baseado no seu cronograma atual.}
    }
}

📅 AGENDAMENTOS{
    {Como funciona o processo de agendamento?
        {O cliente localiza o profissional, escolhe o serviço e inicia a Solicitação. Se o profissional estiver livre e topar as condições, ele aceita e ali o serviço entra na fase ativa no sistema.}
    }
    {Posso cancelar um agendamento? Qual a política de cancelamento?
        {Sim, agendamentos já fechados podem ser cancelados através da própria tela da solicitação na plataforma. É recomendável informar a outra parte via mensagem como forma de empatia.}
    }
    {Como remarco um serviço já agendado?
        {A melhor forma é entrar em contato direto via chat do projeto com o prestador e definirem em comum acordo a nova data.}
    }
    {Onde vejo meus agendamentos como cliente?
        {Acessando o seu Dashboard, você encontra a seção 'Solicitações' ou 'Agendamentos' que centraliza todos os profissionais que você contratou recentemente.}
    }
    {Onde vejo os agendamentos recebidos como profissional?
        {Também pelo Dashboard, mas do lado de 'Solicitações Recebidas', onde você pode gerenciar os clientes, as mensagens e o status do fluxo de trabalho.}
    }
    {Como confirmo que o serviço foi concluído?
        {Ao terminar o trabalho, existe o botão dentro da Solicitação para marcar o status de trabalho como 'Concluído', ativando em seguida o sistema de avaliações.}
    }
}

💰 FINANCEIRO E PAGAMENTOS{
    {Como funciona o pagamento na plataforma?
        {Nossa plataforma gerencia e intermedeia a sua carteira digital para transações formais cadastradas no fluxo de serviços do sistema.}
    }
    {Quando recebo o dinheiro pelo serviço prestado?
        {Isso é liberado normalmente após a finalização/conclusão do serviço dentro da plataforma, indo de acordo com o saldo e carteira do projeto.}
    }
    {Qual a taxa que a plataforma cobra?
        {Pesquisa e cadastro não possuem taxas. Taxas transacionais apenas ocorrem pela operação e administração financeira nos fechamentos de serviços.}
    }
    {Como cadastro minha conta bancária para receber?
        {Vá no Dashboard em recursos de pagamento/Configurações (como PIX ou Dados Bancários) para cadastrar por onde realizará seus saques.}
    }
    {O que é "Saldo Pendente"?
        {São quantias travadas referentes a serviços que já começaram ou que estão aguardando o cliente dar o 'ok' e finalizar a contratação.}
    }
    {Como faço para sacar meu dinheiro?
        {Se houver saldo disponível na carteira, no painel financeiro haverá a opção de efetuar saque para sua conta cadastrada.}
    }
    {Existe valor mínimo para saque?
        {As exigências de saque, por questões de taxas de transferência da rede, estarão destacadas na sua janela de Solicitação de Saque.}
    }
    {Quanto tempo demora o saque para cair na conta?
        {Geralmente leva os prazos de compensação padrão bancário ou processadores estipulados (normalmente entre 1 a 3 dias úteis).}
    }
    {Como vejomeu histórico de transações?
        {Há uma área de painel financeiro / Transações, contendo entradas, retenções e saídas transparentemente contabilizadas.}
    }
}

⭐ AVALIAÇÕES E REPUTAÇÃO{
    {Como funciona o sistema de avaliações?
        {A avaliação mútua é aberta apenas após a formalização do encerramento de uma solicitação; cada lado atribui nota de estrelas e comentários.}
    }
    {Quando posso avaliar um profissional?
        {Somente após o serviço do contrato atuar como "Concluído" pelas partes no painel.}
    }
    {O profissional também pode me avaliar?
        {Sim! Profissionais também avaliam comportamentos dos clientes. Esse é nosso diferencial de verificação e segurança da comunidade.}
    }
    {Posso editar ou remover uma avaliação?
        {Visando a credibilidade da WhoDo!, as avaliações são definitivas. Só removemos em evidência de fraude ou ofensa ao nos contactar pelo suporte.}
    }
    {Como denuncio uma avaliação falsa ou ofensiva?
        {Você deve apontar no suporte qual avaliação lhe ofende para iniciarmos uma investigação interna contra calúnias.}
    }
    {A avaliação média afeta minha visibilidade na busca?
        {Grandemente. O topo da busca favorece sempre profissionais validados com boas avaliações consecutivas.}
    }
}

    }
    {Posso enviar fotos ou arquivos pelo chat?

🔒 SEGURANÇA E PRIVACIDADE{
    {Meus dados pessoais são protegidos?
        {Sua segurança e a obediência e LGPD Brasileira formam nossa base matriz de ética. Dados não são comercializados!}
    }
    {Quem pode ver meu telefone e endereço?
        {Serão restritos salvo indicação pública consciente ou nas interações de confirmação de compromisso fechado entre o prestador.}
    }
    {Como denuncio um usuário suspeito?
        {Reporte e indique diretamente o perfil ou o histórico de mensagens pelo painel de ajuda da plataforma para que banimentos devidos possam agir.}
    }
    {O que acontece se eu for vítima de golpe?
        {Utilizando as travas de transação em plataformas é seguro. Caso feche pagamentos por fora usando ferramentas de busca de rede da plataforma não disporemos do apoio transacional original da nossa responsabilidade final.}
    }
    {A plataforma se responsabiliza pelos serviços?
        {A WhoDo! é a plataforma organizadora e conectora do mercado de trabalho. Apesar de fazermos o sistema limpo de golpistas, a execução é da autonomia técnica dos contratados.}
    }
